GOAL OF THE APP

"Lapha" is the Zulu word for "here" or a word to state the location of a person or thing. Just like the name, the goal of the app is to allow users to monitor what is dear to them, using the application users can monitor their surroundings using their mobile phone as a security camera, with little technical know how.

Lapha can be used as an easy to setup mobile security camera.

The application will provide the following core functionalities:

  • An interface that allows a user to pair their device with another device, one as the camera, the other as the viewer.
  • The application will incorporate AI to perform realtime intrusion detection.
  • The application will allow configuration of notifications based on predefined events.
