TECHNOLOGIES USED

Listed below are the technologies required to build the application

JavaScript/HTML/CSS

JavaScript, HTML and CSS are the underlying technologies that would be used to build the application.

Ionic Vue

Ionic is an open source UI toolkit for building performant, high-quality mobile apps using web technologies. Ionic and integration with the Vue.js frontend framework would be used to implement the core functions of the application and its interface.

WebRTC/WebSocket

The WebRTC protocol enables Real-Time Communications (RTC) in the browser. Due to the application's web-based architecture, WebRTC will be used to facilitate video streaming. In addition, WebSocket will be used to enable two-way communication between the Capture Devand the Gurdiandian Device.

Teachable Machine

The Teachable Machine tool makes building machine learning models fast, easy. It would be used to create the machine learning models required by the application.

TensorFlowJS

JavaScript library TensorFlow.js allows machine learning models to be trained and deployed using JavaScript on the web. Using TensorFlow.js, machine learning models generated from Teachable machine would be run within the application.
